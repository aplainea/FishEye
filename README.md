# FishEye

OpenClassrooms : Projet 6 - FishEye, plateforme de photographes

Compilation SASS en local

```
DEV:
npm run sass-dev

PROD:
npm run sass-prod
```

Pour démarrer le projet sur votre navigateur, il suffit de passer directement par index.html (sur VScode, vous pouvez utiliser "Live Server")
